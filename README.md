# CodeAlpha Task 1: Language Translation Tool

Python application that translates text between 100+ languages using Google Translate API.

## Features
- Real-time text translation
- Support for 100+ languages 
- Interactive ipywidgets GUI for Jupyter Notebook
- Simple dropdown selection for languages

## How to Run
1. Install requirements: `pip install googletrans==4.0.0-rc1 ipywidgets`
2. Open Jupyter Notebook: `jupyter notebook`
3. Run all cells in `CodeAlpha_Task1_Language_Translator.py`
